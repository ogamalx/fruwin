# Sample ISO Test Notes

Use this file to jot down which ISOs you tested fruWin with and what happened.

Example format:

- ISO: ubuntu-22.04-desktop-amd64.iso
  - Result: OK
  - Notes: kernel/initrd under `casper/`, squashfs in `casper/filesystem.squashfs`.

- ISO: archlinux-YYYY.MM.DD-x86_64.iso
  - Result: OK / FAIL
  - Notes: kernel under `arch/boot/x86_64/`, squashfs under `arch/x86_64/`.
